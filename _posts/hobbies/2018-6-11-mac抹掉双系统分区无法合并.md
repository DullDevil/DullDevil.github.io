---
title: mac抹掉双系统分区无法合并
categories: [略知一二]
---

MAC安装win双系统之后，如果没有用启动助理删除win而是直接抹掉的话，会发现用来安装win的分区无法合并。


1、保证win分区的格式为 exfat


2、查看磁盘信息

```
$ diskutil list

/dev/disk0 (internal, physical):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:      GUID_partition_scheme                        *1.0 TB     disk0
   1:                        EFI EFI                     209.7 MB   disk0s1
   2:                 Apple_APFS Container disk2         700.6 GB   disk0s2
   3:         Microsoft Reserved                         16.8 MB    disk0s3
   4:       Microsoft Basic Data 未命名                  299.2 GB   disk0s4
   5:                 Apple_Boot                         134.2 MB   disk0s5

/dev/disk2 (synthesized):
   #:                       TYPE NAME                    SIZE       IDENTIFIER
   0:      APFS Container Scheme -                      +700.6 GB   disk2
                                 Physical Store disk0s2
   1:                APFS Volume MAC OS  HDD             15.9 GB    disk2s1
   2:                APFS Volume Preboot                 44.4 MB    disk2s2
   3:                APFS Volume Recovery                1.0 GB     disk2s3
   4:                APFS Volume VM                      4.3 GB     disk2s4

```
/dev/disk0 下的 3和4 为需要合并的win分区

3、抹掉win对应磁盘

```
sudo diskutil eraseVolume JHFS+ xporter /dev/disk0s4
sudo diskutil eraseVolume JHFS+ xporter /dev/disk0s3
```

xporter：为抹掉之后的分区名，可任意填写

4、合并分区

抹掉之后就可以用磁盘工具对分区进行合并了
